Introducción

Bienvenidos al Proyecto Helsreach. El objeto del mismo consiste en elaborar una herramienta que permita llevar a cabo una integración entre las Shape Expressions y UML, otorgando al usuario la capacidad de generar el equivalente en UML del conjunto deseado de ShEx, o viceversa. Con este propósito, esta aplicación web se organiza del modo que a continuación se describe.

Disponemos de dos secciones análogas para sendas transformaciones entre ShEx y XMI, siendo XMI el formato empleado para representar UML, el cual es apto para la importación en multitud de herramientas profesionales. El usuario podrá aquí aportar su entrada y obtener la correspondiente salida, siendo capaz de modificar a su antojo ambas mediante el editor integrado.

De cara a facilitar una mayor comprensión y usabilidad respecto a las salidas obtenidas, posibilitamos una funcionalidad de generación de UML -para visualizar los XMI que se deseen- así como de generación del grafo correspondiente a un conjunto de Shape Expressions.

Si el usuario no está familiarizado con las Shape Expressions o con XMI, proporcionamos una serie de ejemplos de casuística variada para ayudar en la comprensión de las mismas y su correspondencia con su contraparte.

Por último, el usuario puede personalizar determinados aspectos de la configuración de la página.
